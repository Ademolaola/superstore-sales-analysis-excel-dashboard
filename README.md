# Superstore Sales Dashboard (Excel Project)

## Project Overview
This project analyzes the Superstore sales dataset using Microsoft Excel.
The goal was to clean the data, perform business analysis using PivotTables, and build an interactive dashboard that provides insights into sales performance, profitability, customer segments, and regional trends.

## Tools Used
* Microsoft Excel
* PivotTables
* PivotCharts
* Slicers
* Data Cleaning Functions
  
## Project Steps
1. Converted raw data into an Excel Table
2. Cleaned date columns and corrected text-formatted dates
3. Created delivery time column using order and ship dates
4. Extracted year and month for trend analysis
5. Formatted sales and profit as currency
6. Built PivotTables to answer business questions
7. Designed a dashboard with KPIs and charts
   
## Key Business Questions Answered
* Which product category generates the most sales?
* Which category is the most profitable?
* Which regions perform best?
* How do sales change over time?
* Which customer segment spends the most?
* Do discounts reduce profit?
* Which products sell the most?

## Dashboard KPIs
* Total Sales
* Total Profit
* Total Customers
* Regional Performance
* Sales Trend by Month and Year
  
## Key Insights
* West region generated the highest sales and profit
* Consumer segment contributed the most sales
* Some high-discount products reduced profitability
* Sales showed clear growth trends over time
